Примечания:

  1. Для Использовании скриптов в CSS таблицах используют модифицированные таблицы SASS/SCSS.
  2. Файл с такими инструкциями должен иметь формат соотвтетственно .scss или .sass.
  3. Браузеры не умеют читать стили в таких форматах. Скриптованный файл необходимо преобразовать в обыкновенный css
  4. Это можно сделать двумя способами (оба описаны в первой ссылке):
    Первый - с использованием командной строки и предустановленных пакетов. Соответственно необходимо установить пакет на систему (для разных систем разные способы). P.S. на винде у меня не вышло
    Второй - установить программу, которая будет это делать сама.
  5. Удобная и бесплатная программа - koala (ссылка на скачивание ниже).
  6. После создания скрипта нужно запустить программу, выбрать папку с файлами (директория сохраняется в истории) и нажать на файл и не изменяя настройки на кнопку Compile. Программа самостоятельно найдет все скриптованный файлы и преобразует их в .css формат.
  7. Не забудь импортировать все .css документы в основном шаблоне.
  8. При изменении .sass / .scss документа koala будет автоматически вносить изменения в .css документ.
  9. Иногда адрес сохранения .css ломается. Тогда можно создать файл со своим названием и указать его для компиляции. В данном примере sass_static.scss компилируется в sass.css
